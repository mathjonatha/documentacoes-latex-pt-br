\newcommand*{\GuideBibPreamble}{%
  A seguir, você encontrará muitas referências. Todas elas são referenciadas
  no texto principal. Em muitos casos, a referência aponta para documentos ou
  diretórios que podem ser acessados pela Internet. Nesses casos, a
  referência inclui uma URL em vez de uma editora. Se a referência aponta para
  um pacote {\LaTeX}, então a URL é escrita na forma
  ``\url{CTAN://}\emph{destino}''. O prefixo ``\url{CTAN://}'' significa o
  arquivo \TeX{} em um servidor ou espelho CTAN. Por exemplo, você pode substituir o
  prefixo por \url{https://mirror.ctan.org/}. Para pacotes {\LaTeX}, também é
  importante mencionar que tentamos fornecer um número de versão apropriado
  para o texto que cita a referência. Mas para alguns pacotes é muito
  difícil encontrar um número de versão e uma data de lançamento consistentes.
  Além disso, a versão fornecida nem sempre é a versão atual. Se você
  quiser instalar novos pacotes, certifique-se de que o pacote seja a versão
  mais atualizada e verifique primeiro se o pacote já está disponível em seu sistema.%
}
